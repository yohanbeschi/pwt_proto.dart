isVisible()/isDisplayed()

<div id="box1" style="display:none"></div>
<div id="box2"></div>
<div id="box3" style="height:0px"></div>
<div id="box4" style="visibility:hidden"></div>
<div id="box5">Something inside</div>
Selector: 
--bool isVisible() 
 
$('#box4').isVisible() // Check if the element with id 'box4' has the property 'visibility' set to 'hidden' or not
                       // If ${} return a list, only the first element is checked

--bool idDisplayed() 
 
$('#box2').idDisplayed() // Check if the element with id 'box4' has the property 'display' set to 'none' or not
                         // and if the property 'offsetHeight' is greater than 0 or not.
                         // If ${} return a list, only the first element is checked

Important note : If the element with id 'box5' were to be included in an ascendant element with the property 'display' set to 'none', the method $('#box5').idDisplayed() would return false.
